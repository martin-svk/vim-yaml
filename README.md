Vim-YAML
---

## Info

This is maintained fork of [vim-yaml](https://github.com/avakhov/vim-yaml)
an indent plugin for Vim to fix indentation for YAML files.

## Install

```VimL
Plug 'martin-svk/vim-yaml' " vim-plug
Plugin 'martin-svk/vim-yaml' " Vundle
```
